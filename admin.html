How to use Participants App

  1. Create an event — click "+ New event", fill in details including days and optional MEL question. An event code is auto-generated or you can set your own.
  2. Share the registration link — copy or open the registration link from the event card. Share it with participants before the event so they can register and get a code.
  3. Share the participant view link — share this with program staff. They see the participant list and can open the reg form but cannot access the event admin.
  4. On event day — walk-ins register via the reg form. Everyone gets a unique code to sign attendance.
  5. View & export — use "View participants" to see registrations, search by name or code, and export CSV.
